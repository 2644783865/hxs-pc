# 慧学僧自助打印收款助手

#### 介绍
该项目为Windows系统下的慧学僧自助打印收款助手桌面程序。
慧学僧自助打印收款助手主要功能是协助打印店、图文店收款打印。暂时只上线的功能为微信收款。

#### 软件架构
1. ASP.NET Form
2. Framework4.0
3. 前后端分离
4. 数据校验

#### 安装教程

1. 开发环境VS2010
2. 还原Nuget
3. 目标平台 x86
4. DEBUG模式生成
